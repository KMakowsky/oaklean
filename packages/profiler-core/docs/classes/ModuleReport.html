getCPUUsageFromFile

  • getCPUUsageFromFile(projectReportFilePath, absoluteFilePath): undefined | SourceFileMetaData

  • Returns the CPUUsage of a file

    since a CPUUsage file is anonymized (no absolute path is stored) there is a total of three paths necessary to retrieve the correct CPUUsage of a file

    this.relativeRootDir: describes the location of the execution (relative to the CPUUsage file)

    projectReportFilePath: is the current location of the ProjectReport file

    relativeFilePath: is the location of a measured source file, relative to the execution path (this.relativeRootDir)

    All CPUUsage measurements are stored in this.node_internal, this,intern and this,extern the keys are the corresponding relativeFilePaths

    the absolute path of the measured source file is: path.join(projectReportFilePath, this.relativeRootDir, relativeFilePath)

    therefore the relativeFilePath of the absolute one is: const relativeFilePath = path.relative(path.join(projectReportFilePath, this.relativeRootDir), absoluteFilePath)

    Parameters

    • projectReportFilePath: UnifiedPath

      is the current location of the ProjectReport file

    • absoluteFilePath: UnifiedPath

      is the absolute path of the measured source file

    Returns undefined | SourceFileMetaData
